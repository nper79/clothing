# Como resolver erro de espaço do Chrome

## Erro: FILE_ERROR_NO_SPACE

Este erro impede o app de funcionar corretamente.

## Solução Rápida (2 minutos):

### 1. Limpar cache do Chrome:
- Pressiona **Ctrl + Shift + Delete**
- Seleciona:
  - ✅ Cached images and files
  - ✅ Cookies and other site data
- Time range: **All time**
- Clica **"Clear data"**

### 2. Limpar dados específicos:
- Vai para **chrome://settings/privacy**
- Clica em **"Clear browsing data"**
- Seleciona **"Advanced"**
- Marca tudo e limpa

### 3. Reiniciar navegador:
- Fecha todas as abas
- Reabre o Chrome
- Recarrega a app

## Se não funcionar:
Tenta usar **modo anônimo** ou **outro navegador** (Firefox, Edge).

## Teste depois:
```bash
npm run dev
```

A app deve carregar sem erros!
