# Como limpar cache do Chrome para resolver erro de espaço

## Erro: FILE_ERROR_NO_SPACE

Este erro indica que o Chrome está sem espaço em disco no cache local.

## Soluções:

### Opção 1: Limpar cache do Chrome (Recomendado)
1. Abre Chrome
2. Pressiona Ctrl + Shift + Delete
3. Selecciona:
   - Time range: All time
   - Cached images and files
   - Cookies and other site data (opcional)
4. Clica "Clear data"

### Opção 2: Limpar dados do navegador específico
1. Vai para chrome://settings/privacy
2. Clica em "Clear browsing data"
3. Selecciona "Advanced"
4. Marca todas as opções e limpa

### Opção 3: Usar Developer Tools
1. Abre Developer Tools (F12)
2. Vai para Application tab
3. Storage → Clear site data
4. Recarrega a página

## Teste depois de limpar:
- Fecha todas as abas do Chrome
- Reabre o navegador
- Tenta carregar a tua app novamente
